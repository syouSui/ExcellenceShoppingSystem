# ExcellenceShoppingSystem


> **项目开发中！**

## 简介

一个前后端分离的购物网站系统后台, 前端采用vue+elementUI开发, 后端采用MVC分层架构。


接口文档详见

## 技术栈
### 前端
+ vue.js
+ elementUI
+ jquery
+ bootstrap
+ echarts绘图
+ easy-pie-chart
+ dynamic-table
+ external-dragging-calendar
+ ...

### 后端
+ maven 管理依赖
+ spring-bean BeanUtils 简化对象封装
+ Jackson 解析与生成 json
+ 数据库访问层采用 Apache DbUtils
+ redis 作为数据库中间缓存( 待实现 )
+ c3p0 作为数据库连接池，优化访问
+ junit 单元测试
+ ...
